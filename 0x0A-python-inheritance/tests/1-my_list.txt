DOCTESTS FOR THE '1-my_list.py' MODULE
======================================

Description: This file contains interactive
    doctests for the module. The module is
    expected to define a class that inherits
    from the list class and contains a function
    that prints sorted lists


IMPORT REQUIRED MODULES
-----------------------

>>> MyList = __import__("1-my_list").MyList


CHECKING EMPTY LIST
-------------------
>>> l = MyList()
>>> print(l)
[]

Checking if inheritance occured
-------------------------------
>>> l = MyList()
>>> isinstance(l, list)
True

>>> isinstance(l, MyList)
True

>>> issubclass(MyList, list)
True

Checking List of positive unordered ints
----–-----------------------------------
>>> l = MyList()
>>> l.append(4)
>>> l.append(1)
>>> l.append(5)
>>> l.append(3)
>>> l.append(2)
>>> print(l)
[4, 1, 5, 3, 2]

>>> l.print_sorted()
>>> print(l)
[1, 2, 3, 4, 5]


Checking List of negative unordered ints
----------------------------------------
>>> l = MyList()
>>> l.append(-3)
>>> l.append(-8)
>>> l.append(-1)
>>> print(l)
[-3, -8, -1]

>>> l.print_sorted()
>>> print(l)
[-8, -3, -1]


Checking mixture of positive and negative ints
----------------------------------------------
>>> l = MyList()
>>> l.append(6)
>>> l.append(2)
>>> l.append(-3)
>>> l.append(0)
>>> l.append(-10)
>>> print(l)
[6, 2, -3, 0, -10]

>>> l.print_sorted()
>>> print(l)
[-10, -3, 0, 2, 6]


Checking list with repeating ints
---------------------------------
>>> l = MyList()
>>> l.append(4)
>>> l.append(1)
>>> l.append(9)
>>> l.append(4)
>>> l.append(-3)
>>> l.append(4)
>>> l.append(-3)
>>> l.append(9)
>>> print(l)
[4, 1, 9, 4, -3, 4, -3, 9]

>>> l.print_sorted()
>>> print(l)
[-3, -3, 1, 4, 4, 4, 9, 9]
